## Installation

Use the package manager [npm](https://nodejs.org/en/) to install dependencies.

```bash
npm install
```

## Change firebase configurations to your own.

```javascript

const firebaseConfig = {
  apiKey: '',
  authDomain: '',
  databaseURL: '',
  projectId: '',
  storageBucket: '',
  messagingSenderId: '',
  appId: '',
  measurementId: ''
};
```
